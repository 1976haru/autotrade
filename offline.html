📡 오프라인 상태

백엔드 연결이 없어 실시간 계좌 / 주문 / Agent 결정 데이터를 가져올 수 없습니다.

⚠ 오프라인에서는 주문 / 승인 / 봇 시작 / Kill Switch 토글이 동작하지 않습니다. 화면에 보이는 데이터는 마지막 캐시된 정적 자산만 반영됩니다 — 실제 잔고 / 미체결 / 포지션과 다를 수 있습니다.

확인해 보세요

  • 스마트폰 데이터 / Wi-Fi가 켜져 있는지
  • 로컬 backend(uvicorn app.main:app)가 실행 중인지
  • 같은 네트워크에 PC가 연결되어 있는지
  • Tailscale / VPN이 필요한 경로면 연결되어 있는지

※ 본 PWA는 *관제 대시보드*입니다. 실제 매매 / 주문 / Secret 입력 기능은 백엔드 연결이 필요하며, 푸시 알림은 보안 검토 후 별도 제공 예정입니다.
