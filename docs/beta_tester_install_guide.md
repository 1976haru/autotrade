# 베타테스터 설치 가이드 — Agent Trader v1

> 이 문서는 *컴퓨터 초보 베타테스터* 가 PowerShell / 터미널 / 명령어 없이
> Agent Trader v1 을 PC 에 설치하고 첫 실행까지 마치는 단계를 안내한다.

## 0. 시작하기 전

- Windows 10 또는 11 PC
- 인터넷 연결
- KIS 모의투자 (한국투자증권) 계정 — 모의투자 메뉴에서 발급한 *모의용*
  `App Key` / `App Secret` / `계좌번호`
  - **이번 베타에서는 실거래 계좌를 쓰지 않습니다.** 모의투자만.
  - 모의투자 가입 / 키 발급: 한국투자증권 홈페이지 → "오픈API" 메뉴
- (선택) 스마트폰에서도 PC 의 화면을 보려면 **Tailscale** — [별도
  가이드](tailscale_smartphone_access.md) 참조

---

## 1. 설치파일 다운로드

1. 운영자가 알려준 **GitHub Releases** 페이지를 엽니다.
   - 예: `https://github.com/__ORG__/__REPO__/releases`
2. 가장 위쪽의 *최신 베타* 릴리스를 클릭합니다.
3. `AgentTrader-v1-Setup-1.0.0.exe` (또는 `.msi`) 파일을 다운로드합니다.
   - 파일 이름은 버전이 올라가면 숫자가 바뀝니다 (예: `1.0.1`, `1.1.0`).
4. **카톡 / 이메일로 받은 임의의 exe 는 절대 실행하지 마세요.** 항상
   GitHub Releases 의 공식 파일만 사용합니다.

---

## 2. AgentTrader-v1-Setup.exe 실행

1. 다운로드한 파일을 더블클릭합니다.
2. Windows 보안 경고가 나오면 아래 §3 처리.
3. 설치 화면이 뜨면:
   - 언어 선택: **한국어**
   - 설치 위치: 기본값 그대로 (`%LOCALAPPDATA%\Agent Trader v1\`)
   - 바탕화면 바로 가기 만들기: ✅ 체크
   - 시작 메뉴에 추가: ✅ 체크
   - **"설치"** 버튼 클릭
4. 1~2분 후 설치 완료.

---

## 3. Windows 보안 경고가 나오면

베타 단계라 **Microsoft SmartScreen** 이 경고를 띄울 수 있습니다.

- 화면에 "Windows에서 PC를 보호했습니다" 가 뜨면:
  1. 작은 글씨로 **"추가 정보"** 클릭
  2. **"실행"** 버튼 클릭
- "이 앱을 실행해도 괜찮을까요?" 가 뜨면 **"예"** 클릭

> 정식 코드 서명 인증서를 받기 전까지는 이 경고가 나옵니다. 운영자가
> 직접 전달한 *GitHub Releases* 파일만 실행하면 안전합니다.

---

## 4. 바탕화면 아이콘 실행

- 바탕화면의 **"Agent Trader v1"** 아이콘을 더블클릭합니다.
- 약 5~10초 후 앱 창이 열립니다.
- 처음 실행할 때 *backend* 가 자동 시작됩니다. 잠시 검은색 콘솔 창이 잠깐
  보일 수 있습니다 — 닫지 마세요.

---

## 5. 첫 실행 설정

앱이 열리면 **"환영합니다 — 첫 실행 설정"** 화면이 표시됩니다.

순서대로 입력:

### 5.1 모드 선택

- **SIMULATION (가짜 데이터)** — 추천. 시장 데이터 / 주문 모두 시뮬레이션.
- **PAPER (실제 시세 + KIS 모의투자 가상 자금)** — 모의 환경.
- *LIVE_* 옵션은 회색으로 비활성 — **현 베타 단계에서 선택 불가**.

### 5.2 KIS 모의투자 정보 입력

- **App Key**: 한국투자증권 모의투자에서 발급한 키
- **App Secret**: 같은 화면에서 발급한 secret. *Secret 표시/숨김* 버튼으로
  값을 확인 가능.
- **계좌번호**: 모의투자 계좌 (보통 8자리-2자리 형식)
- **KIS_IS_PAPER**: ✅ (회색으로 고정, **베타 단계에서 변경 불가**)

> **입력한 정보는 PC 안에만 저장**됩니다. 베타 운영자에게도 전송되지
> 않습니다.

### 5.3 저장 / 연결 테스트

- **"저장"** 버튼 클릭
- **"연결 테스트"** 버튼 클릭 → 1~2초 후 결과 표시
  - ✅ "KIS 모의투자 연결 성공" — 다음 단계로
  - ❌ 실패 시 입력값을 다시 확인. App Secret 은 발급 시 *복사 1회 한정*
    이므로 다시 발급이 필요할 수 있습니다.

## 6. 메인 화면 사용

- 화면 상단 좌측에 **"에이전트 트레이더 v1"** 표시
- 우측의 **버전 배지** (`📦 v1 · 1.0.0`) 를 클릭하면 최신 변경 사항 모달
  열림
- **Dashboard / 전략 / 결재 / 백테스트 / 시장 / 설정** 등 탭으로 탐색

핵심 안전 표시:
- 모든 자동 매매 동작은 *SIMULATION 또는 PAPER*. 실거래는 운영자 별도 옵트인
  전까지 비활성.
- "AI 자동 실행" / "선물 실거래" 같은 위험 모드는 화면에서도 disabled (회색).

PC 의 Agent Trader 화면을 스마트폰에서도 보고 싶다면:

- [`tailscale_smartphone_access.md`](tailscale_smartphone_access.md) 참고.

- 공유기 포트포워딩 / 외부 공개 IP 설정은 **절대 하지 마세요** — 보안 위험.

## 8. 업데이트 확인 버튼

업데이트가 나올 때마다 운영자가 새 exe 를 카톡으로 보내지 *않습니다*.
앱 안에서 직접 확인하세요.

1. `설정` 탭 → **"업데이트 확인"** 카드
2. **"업데이트 확인"** 버튼 클릭
3. 새 버전 있으면 자동 다운로드 진행
4. 끝나면 **"재시작하여 적용"** 버튼이 나타남 → 클릭
5. 앱이 종료되고 새 버전으로 자동 재시작됩니다 (입력한 키 / 설정은 그대로
   유지)

> 만약 *"업데이트 실패"* 가 뜨면:
> - 네트워크 (Wi-Fi) 가 연결돼 있는지 확인
> - 그래도 안 되면 GitHub Releases 의 *최신 installer 를 다시 다운로드*해서
>   기존 위에 덮어쓰면 됩니다. 데이터 / 설정은 보존됩니다.

## 9. 문제가 생기면 — 진단 정보 복사

문제가 생기면 운영자에게 정확히 알려야 빠른 도움을 받을 수 있습니다.

1. `설정` 탭 → **"진단 정보 복사"** 버튼 클릭
2. 클립보드에 *마스킹된* 로그가 복사됩니다 — 자동으로 *App Secret / API Key /
   계좌번호* 가 `***` 로 가려져 있습니다.
3. 운영자에게 카톡 / 이메일로 *복사한 텍스트만* 붙여 넣습니다.

> 진단 정보에는 **버전 / 마지막 50건 에러 메시지 / OS 정보 / 안전 flag 상태**
> 만 포함됩니다. 키 / Secret / 매매 내역은 포함되지 않습니다.

---

## 10. 절대 보내면 안 되는 것 (보안)

베타 운영자라도 **다음 정보를 요청하지 않습니다**. 카톡 / 이메일 / 채팅
어디서든 누군가 요구하면 즉시 보고 후 차단하세요.

🚫 **절대 외부에 보내지 마세요**:

- KIS **App Secret**
- KIS **App Key** (보안상 secret 과 같은 등급으로 취급)
- 증권사 **계좌번호**
- 증권사 / 은행 **인증번호** (OTP / SMS)
- 신분증 사진 / 주민등록번호
- 윈도우 **로그인 비밀번호**

> 운영자는 진단을 위해 *마스킹된 진단 정보* 만 요청합니다. 위 secret 들을
> 묻는 메시지가 오면 *피싱* 입니다.

## 11. 자주 묻는 질문 (FAQ)

**Q. 설치 후 앱이 안 열려요.**
A. 바탕화면 아이콘 → 마우스 오른쪽 → "관리자 권한으로 실행". 그래도 안 되면
진단 정보 복사 (§9).

**Q. 모의투자 키를 어디서 발급하나요?**
A. 한국투자증권 홈페이지 → "오픈API" 메뉴 → "모의투자" → App Key 발급.

**Q. 자동매매가 실제로 주문하나요?**
A. **이번 베타에서는 절대 실거래 주문이 나가지 않습니다.** 모든 모드는
SIMULATION 또는 PAPER (모의투자 가상 자금). 운영자가 LIVE 활성화를 위한 별도
승인 절차를 거치지 않는 한 실거래는 불가능합니다.

**Q. 데이터는 어디에 저장되나요?**
A. `%LOCALAPPDATA%\Agent Trader v1\` 폴더에 저장됩니다. 외부 서버로 전송되지
않습니다.

**Q. 삭제하려면?**
A. Windows 설정 → 앱 → "Agent Trader v1" → 제거. 데이터 폴더
(`%LOCALAPPDATA%\Agent Trader v1\`) 도 함께 삭제하면 완전 제거됩니다.
