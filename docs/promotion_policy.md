# Promotion Policy

전략은 아래 단계를 통과해야만 다음 단계로 승격한다.

| 단계 | 설명 | 주문 가능 여부 | 승격 기준 |
|---|---|---:|---|
| Draft | 아이디어 문서화 | 불가 | 전략 조건·청산조건 명확화 |
| Backtest | 과거 데이터 검증 | 불가 | 거래 100회 이상, 기대값 양수, PF 1.2 이상 |
| Paper | 모의투자 | 모의주문 | 4주 이상, 손실한도 위반 0회 |
| Shadow | 실전 시세 신호 기록 | 불가 | 실시간 데이터 누락/오작동 0건 |
| Live Manual | 사용자 승인 후 실거래 | 승인 필요 | 소액 1~2개월 안정성 확인 |
| AI Assist | AI 후보 제안 + 사용자 승인 | 승인 필요 | AI 추천 정확도·거절 사유 분석 |
| AI Execution | 제한 조건 내 AI 자동실행 | 가능 | 별도 승인, 최소 자본, 엄격한 한도 |

## 금지 기준

- 승률만으로 승격 금지
- 특정 하루의 고수익만으로 승격 금지
- 수수료·세금·슬리피지 미반영 백테스트로 승격 금지
- 로그가 남지 않는 주문 기능 금지
