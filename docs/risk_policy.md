# Risk Policy

## 기본값

| 항목 | 기본값 |
|---|---:|
| 1회 주문 최대 금액 | 1,000,000원 |
| 일일 최대 손실 | 200,000원 |
| 최대 보유 종목 수 | 5개 |
| 종목별 최대 노출 | 1,500,000원 |
| 실거래 허용 | false |
| AI 자동실행 허용 | false |

## 주문 전 체크 게이트

1. Emergency Stop 여부
2. 주문금액 한도
3. 일일 손실한도
4. 현금/포지션 보유 가능 여부
5. 최대 보유 종목 수
6. 종목별 노출 한도
7. 운용모드별 허용 여부
8. AI 실행 권한 여부

## 킬스위치 단계

- Level 1: 신규 매수 중단
- Level 2: 신규 매수 중단 + 미체결 취소
- Level 3: 신규 매수 중단 + 미체결 취소 + 청산 후보 표시
